KanSuke プライバシーポリシー

制定日: 2026年7月14日 / 最終更新日: 2026年7月14日

KanSuke（以下「本アプリ」）は、家族などの少人数でスケジュールを共有するためのアプリです。 本ポリシーは、本アプリの利用にあたって取得する情報と、その取り扱いについて定めます。

1. 提供者

  • 提供者: Yokochi（個人開発）
  • お問い合わせ先: yokochi5123@gmail.com

2. 取得する情報と利用目的

情報 取得方法 利用目的
メールアドレス・表示名 Google / Apple でのサインイン時に、各サービスから提供されます アカウントの識別と、同じカレンダーに参加しているメンバーへの表示
ユーザー ID（Firebase Authentication の UID） アカウント作成時に自動発行されます アカウントの識別、データの所有者・参加者の判定
メンバー識別色 アカウント作成時に自動設定され、設定画面で変更できます 「誰の予定か」を色で判別するための表示
予定の内容（タイトル・日時・メモ・参加者・確定/仮の別・繰り返し設定・リマインド設定） 利用者が本アプリに入力します カレンダー機能の提供と、同じカレンダーのメンバーへの共有
カレンダーの情報（名前・参加メンバー・オーナー） 利用者がカレンダーを作成・招待する際に生成されます カレンダーの表示と、閲覧・編集できる範囲の制御
招待リンクの情報（発行者・有効期限・使用回数） 利用者が招待リンクを発行した際に生成されます 招待リンクによるカレンダーへの参加、および取り消し

招待リンクのトークンそのものはサーバーに保存せず、ハッシュ値のみを保存します。 本アプリは、これらの情報を上記の目的以外には利用しません。

3. 取得しない情報

本アプリは、以下の情報を取得しません。

  • 位置情報、連絡先、写真、カメラ・マイクの内容
  • 広告 ID、およびそれに類するトラッキング用の識別子
  • 利用状況の解析データ・クラッシュレポート（アクセス解析ツールや広告 SDK を組み込んでいません）
  • Google / Apple のパスワード（サインインは各サービス上で行われ、本アプリはパスワードを受け取りません）

取得した情報を広告に利用したり、第三者に販売したりすることは一切ありません。

4. 情報が共有される範囲

本アプリは共有カレンダーです。あるカレンダーに登録された予定の内容と、その参加メンバーの表示名・メールアドレス・識別色は、同じカレンダーに参加している他のメンバーに表示されます。 カレンダーに参加していない利用者からは閲覧できません（サーバー側のアクセス制御で制限しています）。

カレンダーへの参加は、メンバーが発行した招待リンクを受諾した場合にのみ成立します。招待リンクを共有する相手にはご注意ください。

5. 情報の保存先と第三者への提供

取得した情報は、Google が提供する Firebase（Firebase Authentication / Cloud Firestore / Cloud Functions）に保存・処理されます。 これは本アプリの機能提供に必要な範囲での委託であり、Google はデータ処理の委託先として、そのプライバシーポリシーおよびデータ処理条件に従って情報を取り扱います。 データは日本国外のサーバーに保存・処理される場合があります。

これ以外に、法令に基づく開示請求など正当な理由がある場合を除き、利用者の情報を第三者へ提供することはありません。

  • Google プライバシーポリシー: https://policies.google.com/privacy
  • Firebase のデータ処理について: https://firebase.google.com/support/privacy
  • Apple でサインイン: https://www.apple.com/legal/privacy/

6. 保存期間と削除

予定・カレンダーの情報は、利用者が削除するか、アカウントを削除するまで保存されます。 なお、オフラインでの編集を複数の端末間で正しく同期するため、アプリ上で削除した予定は、内部的に「削除済み」として記録された状態でサーバーに残ります。この記録は削除済みの予定として扱われ、アプリの画面には表示されません。

端末上には、オフラインでも表示できるようにデータのキャッシュが保存されます。アプリをアンインストールすると、この端末上のキャッシュは削除されます。

7. アカウントの削除

アプリの設定画面から、ご自身でアカウントを削除できます。誤操作を防ぐため、削除の前に確認と再サインイン（再認証）を求めます。 削除すると、アカウント（Firebase Authentication のユーザー情報）、プロフィール情報、および本人のみが参加するカレンダーとその予定・リマインドが削除されます。

アプリを使えない場合など、メールでの削除依頼も受け付けます。yokochi5123@gmail.com 宛に、サインインに使用しているメールアドレスからご連絡ください。本人確認のうえ、同じ処理でアカウントと関連データを削除します。

複数のメンバーが参加している共有カレンダーに登録された予定は、他のメンバーのカレンダー表示を保つため、そのまま残ります（あなたはそのカレンダーのメンバーから外れます）。

8. 安全管理

通信は暗号化（HTTPS）され、サーバー上のデータへのアクセスは、カレンダーの参加メンバーだけに限定するアクセス制御（Firestore Security Rules）で保護されています。

9. 子どもの情報

本アプリは、家族が予定を共有するためのものです。お子様が利用する場合は、保護者の方の同意と管理のもとでご利用ください。 本アプリは、13 歳未満の利用者から意図的に情報を収集することはありません。

10. 本ポリシーの変更

本ポリシーの内容は、必要に応じて変更することがあります。重要な変更を行う場合は、本ページ上で最終更新日を改めてお知らせします。
